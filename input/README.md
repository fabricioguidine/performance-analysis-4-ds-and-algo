# Input Files

This directory contains input files for the experiments.

## Files

- `entrada.txt` - Input file for experiments containing test sizes

### Format

```
N
value1
value2
...
valueN
```

Where `N` is the number of test cases, and each value represents the number of records to process.

**Example:**
```
5
1000
5000
10000
50000
100000
```
