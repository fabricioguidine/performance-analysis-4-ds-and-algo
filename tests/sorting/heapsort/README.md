# HeapSort Results

This directory contains performance results for the HeapSort algorithm.

## Results Format

Results should be organized by input size and include:
- Number of comparisons
- Number of swaps
- Execution time in milliseconds

## Files

- `analysis.txt` - Analysis results (optional, can be generated from raw data)
- Raw result files as needed

## Usage

Results from `output/saida.txt` (Part I) should be processed and organized here for analysis.
