# B+ Tree (d=2) Results

This directory contains performance results for B+ Tree with minimum degree 2.

## Expected Results

### Insertion Metrics
- Number of comparisons
- Number of insertions
- Execution time (milliseconds)
- Average tree level

### Search Metrics
- Number of comparisons
- Number of searches performed
- Average search time (milliseconds)
