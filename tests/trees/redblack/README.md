# Red-Black Tree Results

This directory contains performance results for the Red-Black Tree implementation.

## Expected Results

Results should be organized by input size and include:

### Insertion Metrics
- Number of comparisons
- Number of rotations
- Execution time (milliseconds)

### Search Metrics
- Number of comparisons
- Number of searches performed
- Average search time (milliseconds)

## Files

- `analysis.txt` - Analysis results (optional)
- Raw result files as needed

## Usage

Results from `output/saidaInsercao.txt` and `output/saidaBusca.txt` should be processed and organized here.
